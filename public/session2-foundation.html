Kiến thức nền tảng trong Java

Mục tiêu bài học

  • Hiểu nền tảng Java: JDK, JRE, JVM, bytecode và cách biên dịch/chạy
  • Nắm cú pháp cơ bản: kiểu dữ liệu, biến, toán tử, điều khiển luồng
  • Thao tác với mảng, chuỗi và phương thức
  • Nắm vững OOP: đóng gói, kế thừa, đa hình, trừu tượng hóa
  • Hiểu ngoại lệ (exceptions), collections, generics
  • Áp dụng quy ước đặt tên, cấu trúc package và tổ chức dự án

1. Tổng quan nền tảng Java

Java là ngôn ngữ biên dịch sang bytecode chạy trên JVM (máy ảo Java) → Ưu điểm: đa nền tảng (Write once, run anywhere).

  • JDK (Java Development Kit): Bộ công cụ phát triển (javac, javadoc, tools...).
  • JRE (Java Runtime Environment): Môi trường chạy (JVM + thư viện chuẩn). Từ Java 11, JRE được gộp trong JDK.
  • JVM: Thực thi bytecode, quản lý bộ nhớ (GC), JIT compilation.

Quy trình biên dịch/chạy cơ bản:

# Biên dịch
javac HelloWorld.java
# Sinh ra HelloWorld.class (bytecode)
# Chạy
java HelloWorld

2. Cấu trúc chương trình Java

Một chương trình Java tối thiểu có một class chứa phương thức public static void main(String[] args).

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Xin chào Java!");
  }
}
  • Tên file trùng với public class (HelloWorld.java).
  • main là điểm bắt đầu khi chạy chương trình.
  • Tuân thủ quy ước tên: ClassUpperCamelCase, method/biến lowerCamelCase, hằng số UPPER_SNAKE_CASE.

  • Nguyên thủy: boolean, byte, short, int, long, float, double, char
  • Tham chiếu: String, mảng, class, interface, record...
  • Wrapper: Boolean, Integer, Long, Double... (tự động boxing/unboxing)

double rate = 3.14;

4. Toán tử và biểu thức

  • Số học: + - * / %; tăng/giảm ++ --
  • So sánh: == != > >= < <= (với đối tượng dùng equals())
  • Logic: && || !; Toán tử ba ngôi: cond ? a : b
  • Gán kết hợp: += -= *= /= %=

7. Phương thức, phạm vi truy cập, static

public class MathUtil {
  public static int sum(int a, int b) { // static: thuộc lớp
    return a + b;                      // overloading: trùng tên khác tham số
  }
  public static int sum(int a, int b, int c) { return a + b + c; }
}

class Demo {
  public static void main(String[] args) {
    System.out.println(MathUtil.sum(1,2)); // gọi trực tiếp qua tên lớp
  }
}
  • public/protected/(package)/private: phạm vi truy cập.
  • static gắn với lớp; thành viên thường gắn với đối tượng (cần new).
  • Tham số truyền theo giá trị (pass-by-value); với tham chiếu, giá trị là tham chiếu.

8. Lập trình hướng đối tượng (OOP)

8.1 Đóng gói

public class Person {
  private String name;           // ẩn dữ liệu
  public Person(String name) { this.name = name; }
  public String getName() { return name; }
  public void setName(String n) { this.name = n; }
}

8.2 Kế thừa và đa hình

class Animal { void speak() { System.out.println("..."); } }
class Dog extends Animal { @Override void speak() { System.out.println("Woof"); } }

Animal a = new Dog(); // đa hình
a.speak();            // gọi phiên bản Dog

8.3 Trừu tượng hóa: abstract class vs interface

abstract class Shape { abstract double area(); }
interface Drawable { void draw(); }

class Circle extends Shape implements Drawable {
  private double r; Circle(double r){ this.r = r; }
  @Override double area(){ return Math.PI * r * r; }
  @Override public void draw(){ System.out.println("Vẽ hình tròn"); }
}

8.4 Package và import

package com.example.app; // đầu file
import java.util.*;      // sử dụng thư viện chuẩn

public class App { /* ... */ }

9. Ngoại lệ (Exceptions)

  • Checked: buộc phải khai báo hoặc bắt (IOException,...)
  • Unchecked: RuntimeException và con (NullPointerException,...)
try {
  int n = Integer.parseInt("123a");
} catch (NumberFormatException e) {
  System.out.println("Chuỗi không hợp lệ: " + e.getMessage());
} finally {
  System.out.println("Luôn chạy");
}

// try-with-resources (tự đóng tài nguyên)
// try (BufferedReader br = Files.newBufferedReader(Path.of("in.txt"))) { ... }

10. Collections và Generics

import java.util.*;

List names = new ArrayList<>(); // generics
names.add("An");
names.add("Bình");
for (String n : names) System.out.println(n);

Map scores = new HashMap<>();
scores.put("An", 10);
scores.put("Bình", 9);
System.out.println(scores.get("An"));

11. Quy ước đặt tên và tổ chức dự án

  • Package theo miền đảo ngược: com.mycompany.project.
  • Mỗi file một public class; tên file trùng tên class.
  • Giữ lớp nhỏ gọn, tách trách nhiệm, viết javadoc khi cần.

12. Bài tập gợi ý

  • Viết lớp Student với thuộc tính, getter/setter, constructor; in danh sách sinh viên.
  • Viết chương trình nhập một mảng số, tính trung bình, tìm min/max.
  • Tạo hệ phân cấp Shape (Circle/Rectangle), tính diện tích và sắp xếp theo diện tích.
  • Viết ví dụ dùng List, Map để đếm tần suất từ trong chuỗi.
