Khóa học bổ trợ — Network Basic & JavaScript Essentials

Mục tiêu

  • Tổng quan các kiến thức nền tảng hỗ trợ môn Lập trình Mạng.
  • Định hướng nội dung ôn tập cho các chứng chỉ: Network Basic, JavaScript Essentials 1 & 2.
  • Gợi ý thực hành/lab và tài nguyên tự học chất lượng.

Các khóa học bổ trợ giúp củng cố nền tảng mạng máy tính và JavaScript — hai mảng quan trọng khi xây dựng ứng dụng mạng từ backend đến frontend.

1. Network Basic (Cisco)

  • Mô hình tham chiếu OSI và TCP/IP; đóng gói dữ liệu, MTU, phân mảnh.
  • Địa chỉ IPv4/IPv6, subnetting, CIDR; định tuyến cơ bản, gateway, NAT.
  • Switching, VLAN, ARP, ICMP; dịch vụ hạ tầng: DHCP, DNS.
  • Phân tích gói với Wireshark: TCP handshake, HTTP request/response, UDP.

Gợi ý thực hành

  • Cấu hình IP tĩnh/động; tính subnet, chia mạng, kiểm tra bằng ping/traceroute.
  • Chụp gói TCP 3-way handshake khi chạy bài TCP/UDP ở các buổi 5–6.
  • Thử NAT trên router ảo (GNS3/Packet Tracer) và quan sát thay đổi địa chỉ.

2. JavaScript Essentials 1 (Cisco)

  • Kiểu dữ liệu, biến (let, const), toán tử, điều khiển luồng.
  • Hàm, phạm vi, mảng, đối tượng; thao tác chuỗi và số.
  • Cơ bản DOM/event (nếu học trên nền web) và thực hành debug.

Lợi ích & Chứng chỉ

  • Củng cố nền tảng JavaScript để đọc/viết tool kiểm thử API, thao tác dữ liệu JSON.
  • Hỗ trợ xây UI đơn gi���n phục vụ demo/đồ án backend mạng.
  • Chứng chỉ: Cisco Skills For All – JavaScript Essentials 1 (miễn phí, trực tuyến).

3. JavaScript Essentials 2 (Cisco)

  • ES6+: arrow function, template literal, destructuring, module import/export.
  • Bất đồng bộ: callback → Promise → async/await; Fetch API cơ bản.
  • Thực hành gọi API, xử lý lỗi, và hiển thị dữ liệu.

Lợi ích & Chứng chỉ

  • Nắm b��t đồng bộ hiện đại để gọi dịch vụ REST/WebSocket, quan trọng ở ứng dụng mạng.
  • Chuẩn bị nền tảng cho framework frontend hoặc tool CLI bằng Node.js.
  • Chứng chỉ: Cisco Skills For All – JavaScript Essentials 2 (miễn phí, trực tuyến).

4. Liên hệ với chương trình học

  • Network Basic ←→ Buổi 2, 4, 5, 6, 8: địa chỉ mạng, TCP/UDP, multicast, phân tích gói.
  • JS Essentials ←→ xây dựng giao diện/console tool kiểm thử dịch vụ (REST/WebSocket) bổ trợ cho backend Java.
  • Đồ án: có thể tạo UI nhỏ (HTML/JS) để kiểm thử server TCP/UDP hoặc REST gateway.

5. Tài nguyên tham khảo

  • Cisco Skills For All: Network Basics; JavaScript Essentials 1 & 2.
  • MDN Web Docs: JavaScript Guide, Fetch API, Async/Await.
  • Wireshark Labs: thực hành bắt gói TCP/UDP/HTTP.
