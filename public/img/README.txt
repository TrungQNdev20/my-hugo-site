This folder should contain static images used by the site.
The theme example references `img/avatar.jpg`.
If you want the example avatar, copy:
  themes\almeida-cv\static\img\avatar.jpg
to:
  static\img\avatar.jpg

You can copy it using Windows CMD:
copy "themes\almeida-cv\static\img\avatar.jpg" "static\img\avatar.jpg"
